## 🚀 Comment Créer vos Catégories

### 1. Accéder au Wiki
Connectez-vous et allez dans la section **Wiki** de l'application.

### 2. Créer une Catégorie
- Cliquez sur le bouton **"Nouvelle Catégorie"**
- Remplissez :
  - **Nom** : Ex: "Support Technique"
  - **Icône** : Ex: 🔧 (emoji de votre choix)
  - **Description** : Ex: "Procédures de support"
  - **Couleur** : Choisissez une couleur

### 3. Créer des Sous-Catégories
- Dans une catégorie, cliquez sur **"Nouvelle Sous-Catégorie"**
- Remplissez les informations similaires

### 4. Créer des Articles
- Dans une sous-catégorie, cliquez sur **"Nouvel Article"**
- Rédigez votre contenu

## 🗑️ Si vous avez déjà des catégories existantes

Si votre base de données contient déjà des catégories par défaut et que vous voulez repartir à zéro :

### Option 1 : Supprimer manuellement via l'interface
- Supprimez chaque catégorie une par une depuis l'interface Web

### Option 2 : Utiliser le script de nettoyage (ATTENTION : Supprime TOUT)
```bash
python clear_wiki_categories.py
```

⚠️ **ATTENTION** : Ce script supprime **TOUTES** les catégories, sous-catégories et articles du Wiki !
