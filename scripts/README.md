# Scripts Utilitaires

Ce dossier contient les scripts utilitaires pour la gestion de Dispatch Manager.

## migrate_db.py

Script de migration de la base de données. Crée les tables nécessaires et ajoute les données par défaut.

```bash
python scripts/migrate_db.py
```

## tech.py

Script de gestion des techniciens. Permet d'ajouter et de modifier les techniciens dans la base de données.

```bash
python scripts/tech.py
```
